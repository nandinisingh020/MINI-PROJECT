#MINIPROJECT
Invisible Cloak Project

Developed a real-time computer vision project using Python and OpenCV that creates an invisibility effect similar to the Harry Potter cloak concept. The system captures the background frame and detects a specific cloak color using HSV color masking techniques. The cloak area is then replaced with the stored background frame to generate the invisible effect in real time.

Features
Real-time webcam processing
Background capture and replacement
HSV color detection and masking
Noise reduction using image processing
Live invisibility effect display


Technologies Used
Python
OpenCV
NumPy

Installation
pip install opencv-python numpy
Run the Project
python invisible_cloak.py

Press q to exit the application.
